## Config File

The **Riku** config file of your project helps us compile your code with ease.

### Options

#### env - `Array<string>`

This is used to generate any `environmental` variables used in production.

<span class="tiny">
  **Note:** This is not a required field in your config file
</span>

---

#### discord

<span style="line-height:2">
  This is an object of settings used when initializing your Discord bot
</span>

<span class="tiny">
  **Note:** This is a __required__ field in your config file
</span>

<CH.Code client:load>

```json Example
{
  "token": "your-discord-token-goes-here",
  "application": {
    "id": "your-application-id-goes-here"
  },
  "options": {
    "intents": []
  }
}
```

</CH.Code>
